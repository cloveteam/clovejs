# Contributors

Thanks to everyone who has contributed to CloveJS!

## Becoming a contributor

Contributions of any kind are welcome — bug reports, documentation, and pull
requests. See [CONTRIBUTING.md](./CONTRIBUTING.md) for how to set up the
project, the checks CI runs, and how to open a pull request. If you'd like
your name added here, include the update alongside your contribution.
